jQuery Liquid Carousel v2.0 plugin

Notice: This plugin is still under development and haven't been fully tested. Please don't use it in a production site!

Liquid carousel is a jQuery plugin intended for liquid designs. Every time the container of the carousel gets resized, the number of items in the list change to fit the new width. Resize this window to see it in action.

Uses css transitions if they are available (fallbacks to jQuery animation)
